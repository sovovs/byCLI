# SMZDM (什么值得买)

**Mode**: 🔐 Browser · **Domain**: `smzdm.com`

## Commands

| Command | Description |
|---------|-------------|
| `bycli smzdm search` | |

## Usage Examples

```bash
# Quick start
bycli smzdm search --limit 5

# JSON output
bycli smzdm search -f json

# Verbose mode
bycli smzdm search -v
```

## Prerequisites

- Chrome running and **logged into** smzdm.com
- [Browser Bridge extension](/guide/browser-bridge) installed
